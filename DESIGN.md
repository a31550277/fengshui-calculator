# 風水看房決策系統 Pro：設計藍圖

## 核心原則

這個系統不把風水簡化成「吉 / 凶」二分，而是拆成五種判斷：

- 硬傷：外局、漏水、噪音、逃生、路沖、中宮廁等不容易改變的問題。
- 可改善：床位、視線、穿堂、門簾、窗簾、除濕、家具配置等可低成本修正的問題。
- 情境差異：買房重視長期與轉手；租房重視短期舒適與退租彈性。
- 流派差異：同一格局在形勢、八宅、玄空、陽宅三要、現代居住裡可能權重不同。
- 資料信心：缺坐向、缺年份、缺平面圖時，不能假裝已經能精準判斷。

## 資料模型

### 決策情境

- `purpose`：買房或租房。
- `horizon`：預計居住年限。
- `homeType`：大樓、公寓、透天、套房、住辦。
- `occupants`：居住人數。
- `primaryGoal`：健康、財務、學業、關係或整體平衡。
- `budgetPressure`：預算壓力，用來決定是否能用價格補償缺點。

### 建物資料

- `buildingFacing`：建物 / 社區主要朝向，偏外局與玄空。
- `buildingFacingDegrees`：建物主要朝向度數。
- `buildingFacingAccuracy`：建物坐向資料精度。
- `unitDoorFacing`：本戶大門朝向，偏八宅、陽宅三要與實際納氣。
- `unitDoorDegrees`：本戶大門度數。
- `unitDoorAccuracy`：本戶大門量測精度。
- `builtYear`：建造或入伙年份。
- `floor`：樓層。
- `totalFloors`：總樓層。
- `condition`：屋況。

### 九宮配置

九宮格以八方加中宮表示：

- 東南、南、西南
- 東、中宮、西
- 東北、北、西北

每宮可標示：

- 大門 / 玄關
- 客廳
- 主臥
- 客臥
- 書房 / 工作區
- 廚房 / 爐灶
- 廁所 / 浴室
- 陽台 / 大窗
- 儲藏 / 空置
- 未知

## 評分架構

基礎分數為 `68`，再依下列項目加減：

- 優勢加分：明堂、靠山、方正、採光通風、主位有靠、安全動線、隱私、收納。
- 風險扣分：路沖、反弓、尖角、噪音、特殊場域、缺角、穿堂、門沖廁、門見灶、樑壓、床沖、中宮廁、濕氣漏水。
- 九宮扣分：廁所落中宮、廚房落坎水、門位與朝向資料矛盾。
- 坐向修正：建物坐向精度主要影響玄空與形勢外局，本戶大門精度主要影響八宅與陽宅三要；沒有量測會降權，專業確認會提高可信度。
- 二十四山修正：建物與本戶大門都可輸入精確度數，系統會粗判二十四山、八方、方位交界與欄位矛盾。
- 三元九運修正：建造或入伙年份會判斷六運、七運、八運、九運，並提示九運南向、八運換運等注意事項。
- 流派權重修正：流派權重不只顯示，也會進入總分。權重越低，該流派規則對總分影響越小。
- 情境修正：買房放大高風險扣分；租房給可改善項目些微彈性。
- 居住條件修正：低樓層、頂樓、套房多人居住等。
- 算法符合度：報告必須標示每個流派目前是初篩、半完整、待正式起盤或資料不足，避免把規則引擎說成完整派盤。

## 流派處理

### 形勢派

最適合做第一層初篩，因為外局、道路、明堂、靠山、形煞與動線相對容易從照片與現場觀察取得。

### 八宅派 / 九宮

目前先以門、主臥、廚房、廁所落宮做簡化判斷。完整版應加入：

- 居住者出生年
- 性別
- 命卦
- 東四命 / 西四命
- 門、主、灶方位配合

### 玄空飛星

目前以建物 / 社區主要坐向為主，只做資料完整度、坐山向首、二十四山粗判、三元九運與資料缺口提醒；不因九運、南向或年份直接判吉凶。完整版應加入：

- 坐山與朝向分開輸入
- 二十四山兼向
- 替卦判斷
- 建造或入伙年份
- 山星、向星、運星
- 流年飛星
- 山水與實際外局校正

### 陽宅三要

以門、主、灶為核心，並觀察廁所、門沖、灶位、主臥安定度。

### 現代居住

用實證生活條件校正傳統判斷：

- 採光
- 通風
- 濕度
- 噪音
- 隱私
- 逃生
- 門禁
- 維護成本
- 冷氣與隔熱
- 轉手接受度

## 理想完整版路線

### 第 1 階段：目前版本

完成互動式表單、九宮格輸入、買租分流、多流派報告與看房清單。

### 第 2 階段：資料外部化

把規則從 `app.js` 抽成 JSON：

- `rules/form.json`
- `rules/eight-mansion.json`
- `rules/flying-star.json`
- `rules/modern.json`
- `rules/remedies.json`

### 第 3 階段：平面圖輸入

加入拖拉式平面圖：

- 使用者可拖曳房間到九宮格。
- 可標示門、窗、床、爐、水槽、廁所、樑柱。
- 可上傳照片並加註。

### 第 4 階段：命卦與飛星

加入真正計算：

- 八宅命卦
- 二十四山
- 三元九運
- 玄空飛星盤
- 流年流月提示

### 第 5 階段：看房比較

支援多物件比較：

- A / B / C 物件總分
- 不可逆風險比較
- 可改善成本比較
- 買房議價理由
- 租房談約注意事項

## 判斷倫理

系統必須明確提醒：

- 風水不能取代驗屋。
- 風水不能取代法規與結構安全。
- 缺資料時要降低信心，不可過度斷言。
- 傳統流派有差異，報告應揭露依據，而不是只給結論。
