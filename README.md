# 風水看房決策系統 Pro

這是新版的風水計算程式原型，設計目標是做成「看房決策輔助系統」，而不是簡單的吉凶清單。

## 使用方式

用瀏覽器開啟：

```powershell
.\feng-shui-calculator-pro\index.html
```

## 新版特色

- 買房與租房採用不同權重：買房重視不可逆缺陷、轉手性與長期維護；租房重視可改善性、生活舒適與退租彈性。
- 加入九宮格輸入：可標示大門、客廳、主臥、廚房、廁所、書房、陽台等位置。
- 分離建物坐向與本戶大門朝向：建物 / 社區主要朝向偏玄空、外局、明堂與整棟納氣；本戶大門朝向偏八宅、陽宅三要、門主灶與實際出入納氣。
- 加入主要居住者出生年與性別：可計算八宅命卦、東四命 / 西四命與四吉四凶方。
- 加入爐口 / 火門朝向：可讓陽宅三要與八宅灶向判斷更完整。
- 加入研究報告建議的規則 profile：換年模式、採向規則、北向基準、宅運依據、玄空版本、紫白版本與二十四山版本會寫入報告。
- 加入流年紫白九星：可依年份產生年度九宮盤，並標出一白、二黑、五黃、八白、九紫等年度提醒。
- 加入多物件比較：可將目前物件加入比較清單，依分數排序並顯示信心、紅旗、優勢、缺資料與決策建議。
- 加入 JSON 匯出 / 匯入：可保存目前輸入、計算摘要、比較清單、規則版本與審計資訊，之後再匯入重算。
- 加入坐向資料完整度：建物坐向與本戶門向各自區分沒有量測、手機粗量、羅盤量測與專業確認，並實際影響對應流派規則權重。
- 加入精確朝向度數：建物與本戶大門都可粗判八方、二十四山、方位交界與朝向欄位是否矛盾。
- 加入建造或入伙年份：判斷三元九運，並在報告中提示八運、九運與換運注意事項。
- 加入流派權重：形勢派、八宅派、玄空飛星、陽宅三要、現代居住都會實際影響總分，而不是只作顯示。
- 報告分成決策結論、重大紅旗、主要優勢、改善建議、流派解析、資料缺口與下次看房清單。
- 報告加入算法符合度：明確標示形勢派、八宅派、玄空飛星、陽宅三要、現代居住目前是「可作初篩」、「半完整」、「待正式起盤」或「資料不足」。
- 總分旁會顯示判斷信心：若物件分數已達 100 分上限，手機羅盤改成羅盤量測可能不再提高總分，但會提高判斷信心，並在「量測精度判讀」中顯示差異。

## 判斷架構

目前採用規則加權模型：

- 形勢派：外局明堂、靠山、路沖、反弓、尖角、噪音與動線。
- 八宅派 / 九宮：以本戶大門為主，判斷門、主臥、爐灶、廁所落宮、中宮問題、方位五行與門向精度降權。
- 八宅派：計算主要居住者命卦，列出四吉方與四凶方，並比對本戶門向、門位、主臥、灶座與爐口。
- 玄空飛星：以建物 / 社區主要坐向為主，目前可產生三元九運、坐山向首、運星盤、山星候選盤與向星候選盤；不會只因九運、南向或年份直接判吉凶，仍需兼向、替卦、三元龍陰陽順逆、山水動靜與流年複核。
- 紫白九星：依流年年份產生年度 overlay，預設以立春換盤；月盤、日盤需下一版接入正式節氣引擎。

## 多物件比較與匯出入

1. 先填寫物件名稱與表單資料。
2. 按「生成專業初篩報告」確認單一物件結果。
3. 按「加入比較」把目前物件加入比較清單。
4. 重複輸入其他物件後，可在比較區查看排序結果。
5. 按「匯出 JSON」保存目前表單、比較清單與規則版本。
6. 按「匯入 JSON」可還原先前匯出的資料。
- 陽宅三要：計算門、主、灶所在宮位、門主同局、門灶五行與爐口朝向。
- 現代居住：採光、通風、濕氣、噪音、安全、隱私、隔熱、維護成本。

## 下一版建議

- 平面圖拖拉：讓使用者直接把房間拖到九宮格。
- 精準坐向：目前已可輸入 0 到 359 度並判斷二十四山；下一步可加入坐山、兼向與替卦。
- 八宅命卦：加入出生年、性別與居住者權重。
- 玄空飛星：加入三元九運、坐山朝向、山星向星、流年飛星。
- 報告匯出：輸出 PDF 或 Markdown，方便看房比較。
- 規則外部化：將規則抽成 JSON，讓不同老師或流派維護自己的判斷庫。

## 重要提醒

此工具適合初篩、比較與看房紀錄，不取代驗屋、法規查核、結構安全、噪音實測、實價登錄分析或專業風水師現場羅盤勘查。目前尚未接入 IANA tzdb、HKO 節氣表、WMM2025 磁偏角與 GeographicLib 測地線，報告會明示這些限制。
